Available Endpoints

  • GET /api/users - Retrieves all users.
  • GET /api/users/:id - Retrieves a specific user by ID.
  • POST /api/users - Creates a new user. Requires JSON data like: { "name": "John", "age": 30 }
  • PUT /api/users/:id - Updates an existing user by ID. Requires JSON data like: { "name": "John", "age": 31 }
  • DELETE /api/users/:id - Deletes a user by ID.
  • GET /api/courses - Retrieves all courses.
  • POST /api/courses - Creates a new course. Requires JSON data like: { "title": "Course Title", "description": "Course Description" }
